# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/e417501e-a467-445f-b65e-16ef8e5934ca

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## تحديث الهيرو السينمائي

تم تحديث الصفحة الرئيسية لتشمل:

- خلفية تعليمية ممتدة بعرض الشاشة.
- ستارة لونية خفيفة تحافظ على تفاصيل الصورة.
- بطاقة زجاجية رئيسية ثلاثية الأبعاد.
- بطاقات إحصائية عائمة ومتجاوبة.
- الحفاظ على وظائف أزرار مصمم الخطة والجلسات الصوتية.
